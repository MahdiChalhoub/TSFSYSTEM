# ⚡ AI COORDINATION - QUICK START

**30-Second Guide to Using Both AIs**

---

## 🎯 **SIMPLE RULE**

```
Code/Terminal/Git → CLAUDE CODE
Everything Else → ANTIGRAVITY
Complex Tasks → BOTH (Antigravity plans → Claude executes)
```

---

## 📋 **CHEAT SHEET**

| I Want To... | Ask This AI |
|--------------|-------------|
| Edit a file | Claude Code |
| Run a command | Claude Code |
| Git commit/push | Claude Code |
| Explain something | Antigravity |
| Plan a feature | Antigravity |
| Long discussion | Antigravity |
| Write docs | Antigravity |
| Build a feature | Antigravity (plan) → Claude (code) |

---

## 💰 **WHY THIS SAVES MONEY**

- **Antigravity**: $0.01/message (cheap!)
- **Claude Code**: $0.35/message (expensive)

**Example**:
- 20-message conversation with Claude = $7
- 20-message conversation with Antigravity = $0.20
- **Savings**: 97%! 💸

---

## 🚀 **3 COMMON WORKFLOWS**

### **1. Quick Fix**
```
You → Claude Code: "Fix bug in file X"
Claude: [fixes it]
Done! (2 messages = $0.70)
```

### **2. Feature Implementation**
```
You → Antigravity: "Design feature X"
Antigravity: [plans, saves to .ai/plans/]

You → Claude Code: "Execute plan: .ai/plans/feature_x.md"
Claude: [implements]
Done! (Antigravity $0.15 + Claude $1.05 = $1.20 vs $7!)
```

### **3. Long Discussion**
```
You → Antigravity: "Let's discuss architecture"
Antigravity: [long conversation]
Done! (20 messages = $0.20 vs $7!)
```
