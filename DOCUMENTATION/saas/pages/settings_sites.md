# Sites & Branches Page

**Path**: `/saas/settings/sites`

## Goal
Manage physical locations (stores, warehouses, offices) for the organization.

## Data Sources
- **Read**: `Site` (via API endpoint `/api/sites`).
- **Write**: `Site` (future: create/edit site).

## User Interaction
- **View**: List of site cards showing name, address, and status.
- **Create**: "New Site" button (placeholder).
